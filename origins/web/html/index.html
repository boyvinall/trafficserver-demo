Apache Traffic Server Cluster

3-Node High-Performance Caching Cluster with Consistent Hashing

Status: Cluster is operational! This page is being served through HAProxy -> ATS -> Origin

Architecture

3 ATS nodes behind HAProxy using consistent hashing. Same URL always hits the same cache node for optimal cache hit rates.

Monitoring

Prometheus collects metrics from all nodes. Grafana provides real-time dashboards showing cache performance.

Testing

Refresh this page multiple times. Check the Via header to see which ATS node served the request.
